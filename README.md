## üéÆ Controls

### Interactive Panel
- **Flow Speed** (0.1 - 5.0): Controls how fast particles move through the field
- **Noise Scale** (0.001 - 0.050): Adjusts the granularity of the flow field pattern
- **Trail Fade** (1 - 50): Changes how quickly particle trails disappear

### Keyboard Shortcuts
- **S**: Save current canvas as PNG image

### Mouse Interaction
- Move your cursor to repel nearby particles (150px radius)
